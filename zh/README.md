# exitvpn.com static website (multi-page)

Pages:
- `index.html` (首頁)
- `about.html` (關於我們)
- `faq.html` (常見問答)
- `privacy.html` (隱私權政策)

## Run locally
```bash
python3 -m http.server 8080
```
Then open: http://localhost:8080

## Customize
- Replace placeholder store links in `index.html`
- Replace `support@exitvpn.com` if needed
- Replace the privacy policy body in `privacy.html` with your official text


Privacy policy content is updated with the provided official text (last updated: 2023-08-14).


## Theme
Default is light mode. Users can toggle to dark mode via the moon/sun button in the header.


## Logo
The site uses `logo-full.png` for the header and footer brand.
